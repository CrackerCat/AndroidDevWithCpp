# Android NDK 开发实践之路

在 Android 开发中使用 JNI 和 NDK 技术。

## JNI 基础准备

JNI 全称：**Java Native Interface**。

它是独立于 Android 的一套接口，可以实现 Java 代码与 C/C++ 代码之间的交互。

打好了 JNI 的基础，才会在 NDK 开发时更加得心应手。
